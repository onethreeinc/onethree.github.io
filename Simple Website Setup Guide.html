Absolutely — since you're after a super minimal personal website (just your image + two social icons), and want it cheap and easy, here’s a step-by-step guide tailored to that:

✅ Overview of the Stack

  • Domain: You already have it via Namecheap — perfect.

  • Hosting: We’ll use GitHub Pages — free and perfect for static sites.

  • Site Content: A single HTML file + your image.

  • Tools: You only need a text editor (like VS Code) and a GitHub account.

🧠 Step-by-Step Guide

Step 1: Create a GitHub account (if you don’t have one)

  • Go to github.com and create an account.

Step 2: Create a new GitHub repository

  • Click the + icon (top-right) > New repository.

  • Name it: yourusername.github.io

  • Set it to public (GitHub Pages only works with public repos for free users).

  • Click Create repository.

Step 3: Prepare your website files locally

  1. Create a folder on your computer, e.g., my-site

  2. Inside, add:

    • index.html – this will be your website

    • Your profile image, e.g., me.jpg

    • Optional: a simple CSS file if you want to style it

Here’s a basic index.html template for your use case:

html
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>Your Name</title>
  <style>
    body {
      background: #000;
      color: #fff;
      text-align: center;
      font-family: sans-serif;
      margin: 0;
      padding: 0;
    }
    img {
      max-width: 200px;
      margin-top: 50px;
      border-radius: 50%;
    }
    .icons {
      margin-top: 20px;
    }
    .icons a {
      margin: 0 10px;
      color: white;
      font-size: 30px;
      text-decoration: none;
    }
  </style>
</head>
<body>
  <img src="me.jpg" alt="My Photo">
  <div class="icons">
    <a href="https://twitter.com/yourusername" target="_blank">🐦</a>
    <a href="https://linkedin.com/in/yourusername" target="_blank">💼</a>
  </div>
</body>
</html>

Just replace the links and image file name.

Step 4: Upload to GitHub

You can do this via:

Option A: Git (if you’re comfortable)

Let me know if you want the commands for this.

Option B: Web upload (easiest)

  1. Go to your new GitHub repo: yourusername.github.io

  2. Click Add file > Upload files

  3. Drag in:

    • index.html

    • me.jpg

  4. Click Commit changes

Step 5: Enable GitHub Pages

  1. In your repo, go to Settings > Pages

  2. Under Source, select Deploy from branch and choose main (or master)

  3. Click Save

GitHub will give you a link like:

arduino
https://yourusername.github.io

Wait 1–2 minutes, then visit it — your site is live!

Step 6: Point your Namecheap domain to GitHub Pages

  1. On GitHub, in your repo, create a file called CNAME (no extension).

    • Inside it, put your custom domain (e.g., yourdomain.com)

    • Commit the file

  2. On Namecheap:

    • Go to Domain List > Manage your domain > Advanced DNS

    • Add 2 A Records:

      makefile
      Host: @
Value: 185.199.108.153
TTL: Automatic

      (Repeat for these IPs: 185.199.109.153, 185.199.110.153, 185.199.111.153)

    • Add a CNAME Record:

      makefile
      Host: www
Value: yourusername.github.io
TTL: Automatic

  3. Wait a few hours — DNS changes take time.
